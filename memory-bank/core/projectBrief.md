# Project Brief - LearnGerman

## Overview
LearnGerman is an interactive web application designed as a comprehensive German language cheat sheet and practice tool. It provides quick-reference tables for the most commonly used German grammar elements.

## Goals
1. Provide clear, well-organized reference tables for German grammar
2. Make navigation intuitive — users should find what they need in 1-2 clicks
3. Store all content in easily editable data files
4. Support self-paced practice through quiz/flashcard modes
5. Work on all devices (mobile, tablet, desktop)

## Target Audience
- German language learners (beginner to intermediate)
- People preparing for German language exams
- Anyone needing a quick German grammar reference

## Core Content Areas
- **Verbs**: Most common verbs with full conjugation tables (present, past, perfect)
- **Pronouns**: Personal, possessive, reflexive, relative pronouns
- **Prepositions**: Akkusativ, Dativ, Genitiv, and Wechselpräpositionen
- **Articles & Cases**: Der/Die/Das rules, case declension tables
- **Sentence Structure**: Word order rules, Nebensatz, questions
- **Common Phrases**: Everyday expressions and their usage
- **Numbers & Time**: Cardinal, ordinal numbers, time expressions

## Success Criteria
- User can find any grammar topic within 2 clicks
- Data files can be edited by non-developers
- App loads fast with no backend dependencies
- Clean, distraction-free UI focused on learning
